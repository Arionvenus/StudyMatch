## Mission 003 - Flexbox & Layouts (In Progress)

### Concepts Learned

- Every HTML element is a box.
- The Box Model:
  - Content
  - Padding
  - Border
  - Margin
- Width
- Height
- Parent vs Child elements
- Block elements
- `display: flex`
- `gap`
- `justify-content`
- `align-items`
- `vh` (Viewport Height)

### Biggest Takeaways

- HTML creates the relationships.
- CSS controls the appearance.
- The parent controls layout.
- The child controls its appearance.
- Flexbox works on the parent.
- You can't center something vertically if there isn't any extra vertical space.
